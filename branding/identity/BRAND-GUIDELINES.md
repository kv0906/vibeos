# vibeOS Brand Guidelines

> The cognitive operating system for builders who ship.

---

## Brand Essence

### Positioning Statement
vibeOS transforms scattered AI chaos into structured flow — turning your AI coding tool into a workspace, knowledge hub, and build engine.

### Brand Promise
**Extension, not tool.** AI as a cognitive extension of you, not a generic assistant.

### Target Audience
- Solo builders shipping with AI
- Product engineers who own problem → code → ship
- Anyone tired of scattered prompts who wants a real system

---

## Core Brand Pillars

| Pillar | Expression |
|--------|------------|
| **Problem-First** | Understand before you build |
| **Simplicity** | Compress complexity, ship clean |
| **Systems Thinking** | Repeatable processes that compound |
| **Ship Culture** | 80% shipped beats 100% planned |

## Brand Personality

### Voice Attributes

| Attribute | Do | Don't |
|-----------|-----|-------|
| **Direct** | Lead with substance | Long preambles, hedging |
| **Confident** | Have conviction | Arrogant, dismissive |
| **Builder-centric** | Speak to makers | Marketing fluff |
| **Human** | Warm but technical | Robotic, cold |

### Tone Examples

```
❌ "We'd be happy to help you supercharge your AI workflow!"

```

## Usage Guidelines

### Logo Usage
- Minimum clear space: 1x height of logomark on all sides
- Minimum size: 24px height for digital, 10mm for print
- Never stretch, rotate, or add effects to the logo

### Color Usage
- Use primary dark palette for main backgrounds
- Accent colors for role differentiation only
- Maintain sufficient contrast (WCAG AA minimum)

### Typography Usage
- Headings: Inter or system sans-serif
- Code/Technical: JetBrains Mono or system monospace
- Body: Inter for readability

---

## Do's and Don'ts

### Do
- Use consistent spacing and alignment
- Maintain the builder-centric voice
- Keep visuals clean and minimal
- Show real workflows, not abstract concepts

### Don't
- Add decorative elements without purpose
- Use generic stock imagery
- Overcomplicate the visual hierarchy
- Deviate from the color palette
