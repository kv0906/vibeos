## Quick Picks by Use Case

**Building a dashboard?**
→ shadcn/ui + Tremor + Recharts + TanStack Table

**Building a landing page?**
→ Astro + Aceternity UI + Magic UI + Tailwind

**Building a form-heavy app?**
→ React Hook Form + Zod + shadcn/ui

**Building something with AI?**
→ Next.js + Vercel AI SDK + 21st.dev

**Need drag and drop?**
→ dnd-kit

**Need a command palette?**
→ cmdk + shadcn/ui

**Building a SaaS?**
→ Next.js + Supabase + Better Auth + Stripe

**Need real-time features?**
→ Convex or Supabase Realtime

**Vietnam market?**
→ Sepay for payments

**Need to send emails?**
→ Resend + React Email

**Deploying a Next.js app?**
→ Vercel (zero config)

**Need a backend with database?**
→ Railway or Render

**Need design inspiration?**
→ Godly (creative), Mobbin (real apps), 21st.dev (AI components)

## Selection Criteria

1. **Zero config preferred** — Works out of the box
2. **Good defaults** — Sensible starting point
3. **Copy-paste friendly** — Own the code
4. **Active community** — Help when stuck
5. **Vibe-code ready** — AI can work with it
