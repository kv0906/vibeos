## Core Principles

### 1. First Principles Over Best Practices

I question assumptions before adopting solutions. "Why?" is my default response to any established practice. I reverse-engineer from desired outcomes, not forward from available tools.

**In practice:**

- Don't copy patterns blindly — understand *why* they exist
- Challenge "industry standards" with "does this fit my context?"
- Seek the root cause, not the surface symptom

### 2. Simplicity as the Destination

Complexity is easy. Simplicity is hard. I accept complexity during exploration but always compress toward simplicity in the final form.

**In practice:**

- Start messy, ship clean
- If it can't be explained simply, I don't understand it yet
- Remove until it breaks, then add back one thing

### 3. Systems Over Goals

Goals are endpoints. Systems are engines. I optimize for repeatable processes that compound, not one-time achievements.

**In practice:**

- Build pipelines, not products
- Automate the repeatable, focus on the creative
- Every manual task is a system waiting to be born

### 4. Leverage Through Tools

I'm not competing with AI — I'm multiplying through it. My edge is taste, judgment, and context engineering. The machines handle execution.

**In practice:**

- Vibe coding > traditional coding
- Context engineering > prompt engineering
- Orchestration > implementation

### 5. Ship and Iterate

Perfection is the enemy of progress. I publish thinking-in-progress because feedback loops beat isolation.

**In practice:**

- 80% shipped beats 100% planned
- Public building creates accountability
- Learn in production, not in theory

## Decision Framework

When facing any decision:

```
1. What's the desired end state? (Outcome)
2. What are the real constraints? (Not assumed ones)
3. What's the simplest path that could work? (First principles)
4. What's the reversibility? (Two-way door vs one-way door)
5. What would I advise a friend? (Remove ego)
```

**Bias toward action** for reversible decisions.  
**Slow down** for irreversible ones.
